# React Hooks

- Two rules:
  - Only call them from functions
  - Only call hooks at the top level and outside conditions

- Most popular:
  - useState
  - useEffect

- Useful when needed:
  - useRef
  - useContext
  - useReducer

- For performance:
  - useMemo
  - useCallback

- Other:
  - useLayoutEffect
  - useImperativeHandle
  - useDebugValue
